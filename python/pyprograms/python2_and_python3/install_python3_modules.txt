StringIO provides a convenient means of working with text in memory using the
file API (read, write. etc.). There are two separate implementations.
The cStringIO version is written in C for speed, while StringIO is written in
Python for portability. Using cStringIO to build large strings can offer
performance savings over some other string conctatenation techniques.

The StringIO and cStringIO modules are gone. Instead, import the io module
and use io.StringIO or io.BytesIO for text and data respectively.
