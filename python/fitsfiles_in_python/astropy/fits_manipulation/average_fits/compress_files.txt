to gzip and gunzip files
==========================

gzip filename.ext     gives filename.ext.gz
gzip -c filenam.ext > anotherfile.gz


or, using pipes
cat filename.ext | gzip > anotherfile.gz


decompressing
=============
gzip -d filename.ext.gz      -d is decompress
gunzip  filename.ext.gz



We can also pipe the decompressed file to the standard output to save it to another file.

gzip -cd filename.ext.gz > anotherfile
gunzip -c filename.ext.gz > anotherfile



for multiple files
==================

gzip *.fits
