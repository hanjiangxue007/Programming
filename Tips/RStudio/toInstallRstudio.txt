To install rstudio in ubuntu
===============================

1. download debian file from:
	https://www.rstudio.com/products/rstudio/download/
	
2. right click, open with software center and install it.
	
	BUT this is old stable version 0.99.491
	
to upgrade to v0.99.858
==========================

1. download debian file from:
	https://www.rstudio.com/products/rstudio/download/preview/
	
	proceed as before
