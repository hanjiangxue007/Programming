vim basics
===========

1. to open txt file using vim editor
vi file_name  (e.g. vi vim.txt)

this will show contents of the file

2. to go back to termial type
type i    to go to insert mode

edit the text 

go to command mode type    esc
save the edited text       :wq

3. vim arrow keys
H = Left	L = right
J = down	K = up
