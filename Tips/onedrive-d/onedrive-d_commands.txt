onedriv-d commands for linux
===============================================================================
onedrive-d --help for commands.

onedrive-d [command]

Commands:
     start    Start the daemon.
     stop     Stop the daemon.
     restart  Stop then start the daemon.
     status   Get the status of the daemon.

e.g., onedrive-d status
